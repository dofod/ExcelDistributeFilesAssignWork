REQUIREMENTS:
python <=2.7
	python.org
xlrd library
	https://pypi.python.org/pypi/xlrd
pywin32 library
	http://www.mirrorservice.org/sites/downloads.sourceforge.net/p/py/pywin32/pywin32/Build%20219/
Java Runtime:
	http://www.oracle.com/technetwork/java/javase/downloads/index.html

If you have java installed, and java not found error occurs,
please check if the path to the Java Runtime is set, For more information:
https://www.java.com/en/download/help/path.xml

1. Download Python - execute "download-python" script in folder
2. Install Python and install other dependancies from the REQUIREMENTS Section
3. Move the excel sheet to the program folder
3. Run the program - execute "run" script
4. Enter Filename ( for example "file.xlsx" for a file named file.xlsx)
5. Enter Column where the files are listed ( for example 'a' for 1st Column )
6. Enter Column where persons are listed ( for example 'b' for 2nd Column )
7. Enter Drive where the files could be located ( for example 'C' for C drive)
8. Script will scan for files and when done for the 1st person, results will be copied to the clipboard
9. Paste the results in the desired location and press enter
